## Requirements

- Node.js v14 LTS or later
- Yarn

## Install

```bash
yarn add @byndyusoft/test-package
```

## Usage

```typescript
import { HelloWorldService } from "@byndyusoft/test-package";

console.log(new HelloWorldService().getHelloWorldMessage());
```
